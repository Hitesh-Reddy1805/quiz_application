Solution for Question 3

Prove the identity: \(1 + \tan^2 x = \sec^2 x\)

Step 1: Start with the left-hand side \(1 + \tan^2 x\).

Step 2: Using the identity \(\tan^2 x = \frac{\sin^2 x}{\cos^2 x}\).

Step 3: Rewrite 1 as \(\frac{\cos^2 x}{\cos^2 x}\).

Step 4: Simplify and use \(\sin^2 x + \cos^2 x = 1\), giving \(\frac{1}{\cos^2 x} = \sec^2 x\).

Conclusion: Identity is proven.
