Trigonometry Quiz

Question 1:

Prove: \((1 - \cos^2 A) \cdot \sec^2 B + \tan^2 B(1 - \sin^2 A) = \sin^2 A + \tan^2 B\)

Solution: Since \(1 - \cos^2 A = \sin^2 A\) and \(1 - \sin^2 A = \cos^2 A\), substituting these into the equation simplifies both sides to \(\sin^2 A + \tan^2 B\), proving the identity.

Question 2:

If \(3 \tan \theta = \sec \theta\), then find the value of \(\cot \theta\).

Solution: Dividing both sides by \(\cos \theta\), we get \(3 \sin \theta = 1\), so \(\sin \theta = \frac{1}{3}\) and \(\cos \theta = \sqrt{1 - \sin^2 \theta} = \frac{2\sqrt{2}}{3}\). Thus, \(\tan \theta = \frac{1}{2\sqrt{2}}\) and \(\cot \theta = \frac{2\sqrt{2}}{1} = 2\sqrt{2}\).
